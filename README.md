**TochIDE** — это современная, быстрая и полностью кастомизируемая среда разработки (IDE), работающая прямо в вашем терминале. Оптимизирована для использования в **Termux** на Android, но отлично работает на любой UNIX-системе.

## ✨ Основные возможности
* **🧱 Трехпанельный интерфейс:** Файловое дерево, мощный редактор и панель виджетов.
* **🧘 Zen Mode (`Ctrl+B`):** Убирайте всё лишнее и фокусируйтесь на коде — окно редактора расширяется плавно и без лагов.
* **🎨 Система тем (`F1`):** Мгновенное переключение между Dark, Light и Custom темами. Легко создать свою в папке `themes/`.
* **🗺️ Миникарта (`Ctrl+G`):** Наглядная структура кода с синхронизированным скроллом.
* **🐚 Встроенный Терминал:** Выполняйте shell-команды не выходя из IDE.
* **📊 Мониторинг ресурсов:** Отслеживайте загрузку CPU и RAM в реальном времени.
* **🔍 Умный поиск:** Быстрая навигация по коду прямо из боковой панели.

## 🛠 Установка и запуск
### 1. Клонируйте репозиторий
```bash
git clone [https://github.com/mintlygit/Toch-IDE-TUI.git](https://github.com/mintlygit/TOCH-IDE-TUI.git)
cd Toch-ide-tui
pip install -r requirememts.txt
```
# Запуск
```
python app.py
```

# ⌨️ Управление (Hotkeys)
## В TochIDE всё управление заточено под клавиатуру. Мы разделили клавиши по категориям для удобства:
🎨 Интерфейс и темы
F1 - Switch Theme Циклическое переключение: Dark ➔ Light ➔ Custom
Ctrl + B - Zen Mode Скрыть/Показать боковые панели (расширение кода)
Ctrl + G Minimap - Переключить видимость карты кода справа
📝 Редактор и Файлы
Ctrl + S - Save Сохранить текущий файл (вспышка в статус-баре)
Ctrl + W - Close Tab Закрыть текущую вкладку с файлом
Ctrl + F - Find Фокус на поле поиска в правой панели
